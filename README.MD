# Trend Visualizer – Fashion-MNIST Analysis

🎨 **AI & Fashion Tech Portfolio Project**  
Analyzing fashion trends using the Fashion-MNIST dataset and visualizing average pixel intensity per class.

---

## 🧰 Technologies & Tools

- **Languages:** Python  
- **Libraries:** NumPy, Matplotlib, TensorFlow (Keras)  
- **Skills Highlighted:** Data analysis, ML dataset preprocessing, visualization

---

## 💡 Project Overview

This project explores trends in clothing items using the Fashion-MNIST dataset.  
It demonstrates AI-driven analysis and visualization techniques relevant to **fashion technology**, providing insights into patterns and trends across categories.
